$ yum install docker

$ vi /etc/sysconfig/docker
GOTRACEBACK='crash' を追記

$ systemctl restart docker
$ systemctl enable docker


$ docker pull cenos:6
centOS6をとってくる

$ docker images
もってるイメージの一覧

$ docker run -it centos:6
コンテナをスタート
Ctrl+p Ctrl+q でコンテナを終了させないで戻る

$ docker attach [ID]
実行中のコンテナに接続

$ docker ps
実行しているコンテナの一覧
-a ですべて

$ docker start [ID]
コンテナをスタート

$ docker stop [ID]
コンテナを停止

$ docker rm [ID]
コンテナを削除

$ docker rmi [ID]
コンテナイメージを削除

$ docker commit [ID] 名前:タグ
コンテナをイメージとして保存

$ docker exec -it [ID] [コマンド]
コンテナ上でコマンドを実行

$ docker logs [ID]
コンテナの標準出力を受け取る

$ docker run -it -p 1000:80 centos:6
ポートを割り当ててコンテナを起動
